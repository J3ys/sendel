Instructions & FAQ

How to Use This App

  1. Search Functionality

    Use the search bar at the top of the page to find specific skills, projects, or certifications. Simply type your query and press Enter or click the search icon.

  2. Filtering Skills

    On the Skills section, use the category buttons to filter skills by domain (Frontend, Backend, Architecture, DevOps, AI).

  3. Viewing Project Details

    Click on the "View Details" button on any project card to see more information about that project, including architecture diagrams and business outcomes.

  4. Dark/Light Mode

    Toggle between dark and light themes by clicking the moon/sun icon in the top navigation bar.

  5. Contact Form

    Fill out the contact form at the bottom of the homepage to send a message. All fields are required.

Frequently Asked Questions

Is this a real portfolio?

This is a template portfolio designed for IT professionals, particularly those specializing in enterprise architecture and e-commerce. You can customize it with your own information.

How do I customize this portfolio?

You can customize this portfolio by editing the HTML, CSS, and JavaScript files. The README.md file contains detailed instructions on how to modify personal information, styling, and images.

Does the contact form actually work?

The contact form in this template is for demonstration purposes only. To make it functional, you would need to add backend code to process form submissions, or connect it to a form service like Formspree.

Can I add more projects or skills?

Yes, you can add as many projects or skills as you want by duplicating the existing HTML structure for project cards or skill cards and updating the content.

Is this portfolio mobile-friendly?

Yes, this portfolio is fully responsive and works well on all devices, including desktops, tablets, and mobile phones.

How do I deploy this portfolio?

This portfolio is designed to be deployed on GitHub Pages. The README.md file contains instructions on how to set up GitHub Pages for your repository.

Can I use this for commercial purposes?

This portfolio template is released under the MIT License, which means you can use it for personal or commercial purposes. However, please check the license details for any limitations.

Troubleshooting

The theme toggle isn't working

If the theme toggle isn't working, make sure JavaScript is enabled in your browser. Also, check the browser console for any errors that might be preventing the script from running properly.

Images aren't loading

If images aren't loading, check that the file paths in the HTML are correct. Make sure the image files exist in the specified directory.

The site looks different in my browser

This site is designed to work with modern browsers. If you're using an older browser, some features may not work as expected. Try updating your browser to the latest version.
